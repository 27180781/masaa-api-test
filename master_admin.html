דשבורד ניהול ראשי - מסע אל תוך עצמי

כלי ניהול

ניהול משחקים (שיוך למייל) ניהול שאלות ניהול תובנות צפייה בתוצאות (דשבורד)

הגדרות גלובליות

תיעוד API וקישורים

דפים למשתמשים

דף תוצאות ללקוח (מפעיל): /results/[מזהה_המשחק]

דף תוצאה אישי למשתתף: /my-result (המשתמש יזין קוד/טלפון)

נקודות API

שליחת תוצאות משחק:

POST /api/submit-results

שליפת תוצאה אישית לפי קוד:

GET /api/my-result/by-code/[קוד_גישה]

שליפת תוצאה אישית לפי טלפון:

GET /api/my-result/by-phone/[מספר_טלפון]
