דשבורד ניהול ראשי - מסע אל תוך עצמי

כלי ניהול

ניהול משחקים (שיוך למייל) ניהול שאלות ניהול תובנות צפייה בתוצאות (דשבורד)

הגדרות גלובליות

תיעוד API וקישורים

דפים למשתמשים

דף תוצאות ללקוח (מפעיל): https://masaa.clicker.co.il/results/[מזהה_המשחק]

דף תוצאה אישי למשתתף: https://masaa.clicker.co.il/my-result

נקודות API

יצירת משחק חדש

POST https://masaa.clicker.co.il/api/games

יוצר שיוך חדש בין מזהה משחק למייל של לקוח.

פורמט JSON לשליחה:

{
    "game_id": "some-unique-id",
    "client_email": "client@example.com"
}

שליחת תוצאות משחק

POST https://masaa.clicker.co.il/api/submit-results

שולח את כל נתוני המשחק הגולמיים לעיבוד.

פורמט JSON לשליחה:

{
    "gameId": "some-unique-id",
    "users": {
        "0501234567": {
            "name": "שם המשתתף",
            "group_name": "שם הקבוצה",
            "answers": { "q1": "1", "q2": "4", ... }
        }
    }
}

שליפת תוצאה אישית לפי קוד

GET https://masaa.clicker.co.il/api/my-result/by-code/[קוד_גישה]

מחזיר את הפרופיל המלא, כולל התובנות, של משתתף ספציפי.

שליפת תוצאה אישית לפי טלפון

GET https://masaa.clicker.co.il/api/my-result/by-phone/[מספר_טלפון]

מחזיר את הפרופיל המלא, כולל התובנות, של משתתף ספציפי.
